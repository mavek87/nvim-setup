# My Neovim Config

## Useful shortcuts:

### Nvim
ctrl + xo ===> autocomplete text (just in insert mode!!!)

## Useful Commands:

- :checkhealt | verify nvim integrity
- :Lazy       | open Lazy packet manager menu
- :Mason      | open Mason menu
- :LspInfo  | print info about language recognized for file
